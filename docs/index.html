JobSpec: The Next Generation

Job Specification for Submitting Computational Work

What is this initiative about?

We are working on an abstract job specification that will take a description of work, including requirements, services, and resources, and transform into the correct submission commands for a specific workload manager or cloud API. We are starting with Flux Framework and will be extending to Kubernetes and other workload managers and APIs soon. Read our Documentation to get started.

How can I contribute?

Check out our projects on GitHub and ping @vsoch for an invitation to the Flux Performance slack (or HPC Containers Slack) to talk about your ideas.
